How to start
------------

1. Create venv.
2. Install requirements.
    ```bash
    pip install -r requirements.txt
    ```
3. Create database and apply migrations.
    ```bash
    python manage.py migrate
    ```
4. Create superuser.
    ```bash
    python manage.py createsuperuser
    ```
5. Run server.
    ```bash
    python manage.py runserver
    ```
